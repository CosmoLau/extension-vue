## 快速开始

需要在 Cocos Creator 项目根目录下的 `extensions` 文件夹中克隆本仓库，如果还没有 `extensions` 文件夹，请先创建一个。

```bash
# 创建 extensions 文件夹
mkdir extensions
# 克隆仓库
git clone https://github.com/CosmoLau/extension-vue.git
```

然后切换到仓库的根目录安装 npm 依赖：

```bash
# 切换到仓库根目录
cd extension-vue
# 安装依赖
npm install
```

完成后，就能在编辑器中通过 `面板` -> `extension-vue` -> `默认面板` 打开示例模板了。
